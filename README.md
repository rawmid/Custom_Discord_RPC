# Discord Custom RPC

A simple desktop application for creating and managing custom **Discord Rich Presence** without writing any code. Customize your activity, images, buttons, and timestamps through an easy-to-use interface.

##  Features

* Custom Rich Presence
* Large & small images
* Interactive buttons
* Timestamps
* Multiple saved presets
* One-click start/stop
* Lightweight and easy to use

##  Getting Started

1. Launch Discord.
2. Run the application.
3. Check Discord Profile

##  About

Built as a fun project to make creating custom Discord Rich Presence quick, simple, and accessible for everyone.
